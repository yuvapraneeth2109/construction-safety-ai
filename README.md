# Construction Safety AI

An AI-based computer vision system that detects safety violations on construction sites.

## Features
- Helmet detection
- Safety vest detection
- Real-time worker monitoring
- Safety violation alerts

## Technologies Used
- Python
- OpenCV
- YOLO
- PyTorch
